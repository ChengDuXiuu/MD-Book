# 项目启动后自动打开浏览器

```js
package.json
"scripts": {
  "serve": "vue-cli-service serve --open",
  "build": "vue-cli-service build",
  "lint": "vue-cli-service lint"
}
```



# 关闭eslint

```js
#src/vue.config.js
module.exports = {
  //关闭eslint
  lintOnSave: false
  }
```



# 根路径别名

```js
#src/jsconfig.json
 {
    "compilerOptions": {
        "baseUrl": "./",
            "paths": {
            "@/*": [
                "src/*"
            ]
        }
    },

    "exclude": [
        "node_modules",
        "dist"
    ]
 }
```
