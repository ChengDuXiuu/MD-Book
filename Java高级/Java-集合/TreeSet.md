## 知识点

*   存放元素顺序 ：无序 ， 但是会按照元素Hash进行排序
*   线程是否安全 ： 不安全
*   数组结构 ：物理结构--链表  |  逻辑结构--二叉树



## 排序原理

*   内部比较器 comparable
*   外部比较器 comparator

>   <font color=ff00aa>如果自定义对象，一定要进行比较器实现</font>
